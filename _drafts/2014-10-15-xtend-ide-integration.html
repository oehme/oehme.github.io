--- layout: post title: Xtend - IDE Integration date: '2014-10-15T11:21:00.000+02:00' author: Stefan Oehme tags: modified_time: '2014-10-15T11:21:40.522+02:00' blogger_id: tag:blogger.com,1999:blog-7973028861743150658.post-4984880229180209798 blogger_orig_url: http://mnmlst-dvlpr.blogspot.com/2014/10/xtend-ide-integration.html --- We strive to make the IDE experience as close as possible to what you are used to from JDT. Below you will find a list of all Xtend IDE features that came to my mind.

We also know that many people out there don't want to use Eclipse or any IDE at all. This is why we started decoupling our code from Eclipse UI dependencies, so we can provide a nice editing experience on other platforms. Currently we are focusing on an Xtend plugin for IntelliJ IDEA. Many of the features listed here will eventually be available there, too.
